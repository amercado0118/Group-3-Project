# Group: 3 Pizza Project
Pizza Ordering System 

**Project Overview:** 

Our project is a Pizza Ordering Sytem that allows users select a pizza size and customize it with the given toppings. Our system focuses on delivering functionality for a ordering system that delivers strong enhancements. 

**How our Program Works**
1. User will select pizza size(S,M,or L).
2. User will select pizza toppings:
    Pepperoni or Extra Cheese
3. Program will convert user input to       uppercase for consistent processing.
4. Price will be calculated.
5. Final total will be displayed and calculated.
